## üöÄ Qo'shimcha Achievements Olish Yo'llari

### 1. Pull Request Yaratish
```bash
# Yangi branch yarating
git checkout -b feature/new-feature

# O'zgarishlar qiling
# ...

# Commit va push
git add .
git commit -m "feat: add new feature"
git push origin feature/new-feature

# GitHub'da Pull Request yarating
```

### 2. Issues Yaratish va Yechish
- Bug report issue yarating
- Feature request issue yarating
- Question issue yarating
- Issue'larni yeching va close qiling

### 3. Discussions Yaratish (Agar yoqilgan bo'lsa)
- GitHub'da Discussions'ni yoqing
- Q&A, Ideas, General discussions yarating

### 4. GitHub Actions Ishlatish
- Workflow'lar avtomatik ishlaydi
- Har bir push/PR'da workflow ishga tushadi
- Workflow'lar muvaffaqiyatli o'tganda achievement olasiz

### 5. Code Review Qilish
- Boshqa PR'larni review qiling
- Comment qo'shing

### 6. Security Advisories

- Security vulnerability topilsa, Security Advisory yarating

### 7. Sponsorships (Agar mavjud bo'lsa)
- Sponsorships yoqing
- Sponsors qo'shing

### 8. Releases Yaratish
```bash

# GitHub'da Release yarating
```

### 9. Wiki Yaratish (Agar kerak bo'lsa)
- Wiki'ni yoqing
- Documentation qo'shing

### 10. Projects va Milestones
- GitHub Projects yarating
