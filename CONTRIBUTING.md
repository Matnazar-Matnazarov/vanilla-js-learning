## ğŸ”€ Pull Request yaratish

### 1. Fork qiling
Repository'ni fork qiling va o'z kompyuteringizga clone qiling:
```bash
git clone https://github.com/YOUR_USERNAME/vanilla-js-learning.git
cd vanilla-js-learning
```

### 2. Branch yarating
Har bir o'zgarish uchun yangi branch yarating:
```bash
git checkout -b feature/your-feature-name
# yoki
git checkout -b fix/your-bug-fix
```

### 3. O'zgarishlar qiling
- Kod yozing
- Test qiling
- README'ni yangilang (agar kerak bo'lsa)

### 4. Commit qiling
Professional commit message yozing:
```bash
git add .
git commit -m "feat: add new feature description"
```

### 5. Push qiling
O'z branch'ingizni push qiling:
```bash
git push origin feature/your-feature-name
```

### 6. Pull Request yarating
GitHub'da Pull Request yarating va template'ni to'ldiring.
